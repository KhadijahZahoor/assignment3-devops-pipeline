# Assignment 3: Kubernetes Orchestration & Full DevOps Pipeline

This project is a 3-tier microservice application built using the DevOps tools covered in the course.

## Architecture

The application contains:

- Nginx reverse proxy
- Flask API backend
- MySQL database

## Tools Used

- Git and GitHub
- Docker
- Docker Compose
- GitHub Actions
- DockerHub
- Kubernetes
- Minikube

## Application Endpoints

- GET /health
- GET /api/items
- POST /api/items
- DELETE /api/items/<id>
